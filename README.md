# Tarea GitHub Colaborativo

## Descripción
Este proyecto consiste en un programa en Python dividido en varios módulos, desarrollado como práctica de trabajo colaborativo en GitHub.

El sistema incluye:
- Un menú principal
- Un módulo de trigonometría
- Un módulo de cuenta cuentos
- Un módulo de contabilidad

## Archivos del proyecto
- `main.py`: menú principal del programa
- `trigonometria.py`: operaciones trigonométricas básicas
- `contacuentos.py`: cuenta palabras en un texto
- `contabilidad.py`: cálculo simple de ingresos, gastos y balance

## Cómo ejecutar
1. Abrir la terminal
2. Ejecutar el siguiente comando:

```bash
python main.py
